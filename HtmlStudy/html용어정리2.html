• 순서를 나타내지 않는 기호 표시
  • 기호를 바꾸려면, css의 라인 스타일 타입에 가서 없애거나, 사각으로 만들거나, 채워진 동그라미나 빈 동그라미로 바꿀 수 있다.
  1. 얘는 타입으로 기호 유형을 영문이나 로마자로 앞의 기호를 바꿀수 있음.
  2. 스타트로 하면 숫자로 시작할수도 있고, 어디부터 카운트할지도 지정 가능.
  3. 타입과 스타트를 동시에 쓴다면 문자타입을 지정 하고, 그 문자의 어디서부터 카운트할지 지정이 가능하다. 예를들어 알파벳타입에 카운트는 d부터 시작한다던가.
  4. reversed로 하면 역순으로도 지정 가능
  5. 밸류 값을 지정하면, 타운트를 해당 라인부터 다시 지정 가능. 스타트로 2부터 카운트 했다가. 밸류로 해당 지점부터 8부터 카운트하는 등.
